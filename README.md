### **Overview:**

It's the same old, familiar WARP Corporation Cleanup Crew, but a little spicier. Since it's designed to fall into similar stat ranges to vanilla, don't expect to have a hard time with Star of the City and above builds when fighting them.

You're free to mix and match the rewards you get from this reception with any other Charge archetype, even the vanilla versions of themselves.

Rose also gains a unique exclusive Combat page that makes it easier to gain a certain Battle Symbol she's wearing... if you use it at the lowest possible Charge.

### **Instructions:**

Subscribe to this mod, then launch the game with the Launch Option "LOR with Mods" enabled, then ensure that this mod is enabled by selecting it in the mod menu when the game loads. You'll know it's enabled if there is a tick mark next to the name.

To invite this reception, make sure that Workshop is ticked on in the Invitation menu, then use a (vanilla) Book of a Puppet.
